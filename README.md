# Automação Dia A Dia

Este projeto tem como objetivo automatizar a preparação de um computador para uso, abrindo automaticamente aplicativos, sites e programas necessários, deixando o ambiente pronto para o usuário iniciar suas atividades sem precisar configurar tudo manualmente.

## Ideal para:

Empresas que preparam máquinas diariamente

Ambientes corporativos

Home office

Setup rápido de estações compartilhadas

Rotinas repetitivas de abertura de sistemas

### Funcionalidades

Abrir aplicativos instalados no computador
Acessar sites automaticamente no navegador
Executar programas específicos
Organizar janelas
Simular teclado e mouse
Inserir logins automáticos (quando permitido)
Automatizar rotinas repetitivas

### Tecnologias Utilizadas

Python 3.x
PyAutoGUI

Bibliotecas auxiliares:

time
subprocess
webbrowser
